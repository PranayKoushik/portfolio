Overview
This is my personal portfolio website built using HTML, CSS, and Font Awesome.
It showcases my professional profile, skills, and projects, with a modern animated design.
The site is fully responsive and optimized for both desktop and mobile devices.

Technologies Used
•	Frontend: HTML5, CSS3
•	Icons: Font Awesome
•	Typography: Dancing Script custom font
•	Animations: CSS keyframes and SVG text effects

Features
• Clean and modern portfolio layout
• Animated SVG “DEVELOPER” text
• Responsive design for all screen sizes
• Circular rotating profile image
• Smooth navigation bar
• Easily customizable content and color themes
